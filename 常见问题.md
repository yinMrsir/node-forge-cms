# 安装nvm卡住怎么办？

使用命令`curl -o- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.7/install.sh | bash`安装时卡住不动。

## 本地脚本安装 nvm

直接复制下面**整段代码**一次性执行，一步到位：

```bash
# 下载 nvm 到本地
git clone https://gitee.com/mirrors/nvm.git ~/.nvm

# 添加环境变量
echo 'export NVM_DIR="$HOME/.nvm"' >> ~/.bashrc
echo '[ -s "$NVM_DIR/nvm.sh" ] && \. "$NVM_DIR/nvm.sh"' >> ~/.bashrc
echo '[ -s "$NVM_DIR/bash_completion" ] && \. "$NVM_DIR/bash_completion"' >> ~/.bashrc

# 立即生效
source ~/.bashrc
```

## 验证安装成功

```bash
nvm -v
```

出现版本号就说明**安装完成**！

# 命令行如何导入数据库？

好！我直接给你**一步到位、复制就能执行**的命令，**创建 `node_forge_cms` 数据库**。

你现在用的是 **Docker 里的 MySQL 8**，密码是：`123456`

---

## 直接复制执行这条命令（创建数据库）

```bash
docker exec -i mysql8 mysql -uroot -p'123456' -e "CREATE DATABASE node_forge_cms;"
```

执行完 **没有报错** 就代表 **创建成功** ✅

---

### 查看是否创建成功（可选）

```bash
docker exec -i mysql8 mysql -uroot -p'123456' -e "SHOW DATABASES;"
```

你会看到列表里有 **node_forge_cms**。

## 一键导入命令（直接复制运行）

```bash
docker exec -i mysql8 mysql -uroot -p'123456' node_forge_cms < /data/www/node-forge-cms/server/node_forge_cms.sql
```

没有任何报错 = 导入成功 ✅

### 验证是否导入成功（查看表）

```bash
docker exec -it mysql8 mysql -uroot -p'123456' node_forge_cms -e "show tables;"
```

能看到一堆表 = 完全成功！

# 重启服务器后 如何自动启动 nginx docker docker 中的mysql8 redis, 和pm2 服务？

我直接给你**最简单、一次配置、永久生效**的方案！
所有服务：**Nginx、Docker、MySQL8、Redis、PM2** 开机 **自动启动**！

## 一、设置 Docker 开机自启（必须先开）

```bash
systemctl enable docker
```

## 二、设置 MySQL8 容器自启

```bash
docker update --restart=always mysql8
```

## 三、设置 Redis 容器自启

```bash
docker update --restart=always redis
```

## 四、设置 Nginx 开机自启

```bash
systemctl enable nginx
```

## 五、设置 PM2 项目开机自启（最重要）

### 1. 生成开机启动配置

```bash
pm2 startup
```

### 2. 保存当前运行的项目

```bash
pm2 save
```

---

## ✅ 全部完成！

现在**服务器重启后**，会**自动按顺序启动**：

1. Docker
2. MySQL8
3. Redis
4. Nginx
5. PM2 项目（你的 node_forge 后台）

---

## 测试是否生效（可选）

```bash
reboot
```

重启后直接访问你的网站，**全部自动运行**！
